JavaScript 30 Project Created By Wes Bos

J30 P1 - Drum Kit

Wes setup a html page template and CSS styling. After watching Wes' video for this project, I wrote JavaScript to play sounds and highlight keys on an event listener. Wes taught this project so that we could setup the keys to be played when the keys were pressed on the keyboard. To get a little more practice, I added event listeners to do the same thing when the "keys" were clicked with a mouse.

J30 P2 - Clock

Starting with CSS & HTML code including the clock built, Wes walked us through adding Javascript to calculate a degrees field for seconds, minutes and hours. That function then updates the transform rotate CSS property to move each hand. To go above what Wes taught, I added an if statement that removed the transition timing for the hands when seconds/minutes/hours are on the 12. Also added some updated CSS styling to the clock hands to make them look a little better, including setting the seconds hand as red.

J30P3 - CSS Styles

Wes provided us with the HTML and CSS (except the CSS variables). After adding the CSS variables, we use Javascript to connect the three inputs to the CSS variables. Those updates happen when you mouseover the inputs and/or when you change an input. After finishing the work Wes presented, I updated the Javascript to include If statements to set CSS variables without the need for custom data attributes on the HTML input elements.
