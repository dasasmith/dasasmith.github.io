JavaScript 30 Project Created By Wes Bos

J30 P6 - Type Ahead

Wes connected the HTML template to a JSON datafile that's a list of the top 1,000 most populated US cities. He walked us through setting up JavaScript that would display a list of options based on what was entered into the Searchbox. After Wes' lesson was done, I added additional JavaScript to force the original prompts to reappear when the searchbox is emptied. I also added JavaScript to tell the user that no matches were found, when that is the case.
